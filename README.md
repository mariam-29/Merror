# 🪞 Merror Script

> *"Write it backwards. Run it forwards."*

Merror Script is a Python-based novelty programming language where every reserved keyword and built-in function is spelled **backwards**. It runs like standard Python — but forces your brain to think in reverse. Designed for coding puzzles, educational fun, and brain-training exercises.

---

## 🔤 The One Rule

**Reverse every keyword and built-in. Everything else stays the same.**

- ✅ Keywords → reversed (`if` → `fi`, `for` → `rof`, `def` → `fed`)
- ✅ Built-ins → reversed (`print` → `tnirp`, `input` → `tupni`, `range` → `egnar`)
- ❌ Variable names → unchanged
- ❌ String contents → unchanged
- ❌ Operators → unchanged
- ❌ Comments → unchanged

## 📐 Language Rules

| Rule | Detail |
|------|--------|
| **Keyword Reversal** | All Python reserved keywords must be written in reverse |
| **Built-in Reversal** | All built-in functions are also reversed |
| **Identifiers** | Variable, function, and class names are NOT reversed |
| **Indentation** | Uses Python-style 4-space indentation — no braces |
| **Operators** | `+`, `-`, `*`, `/`, `==`, etc. remain unchanged |
| **Strings & Values** | String contents and numbers are NOT reversed |
| **Comments** | Use `#` as normal — comment text is NOT reversed |
| **Case Sensitivity** | Merror Script is case-sensitive (`eurT` ≠ `eurt`) |
